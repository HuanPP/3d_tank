# 钢甲狂潮 (Steel Torrent)

钢甲狂潮是一款基于 HTML5 Canvas、Three.js 以及 Vanilla CSS 构建的 3D 第一人称战术坦克模拟对决游戏。游戏采用未来主义的霓虹线框风格，为玩家呈现一个充满速度与对抗的虚拟机甲战场。玩家将置身于高科技坦克驾驶舱中，利用多种先进武器系统与智能敌机展开周旋与厮杀。

## 核心战术系统

游戏集成了先进的座舱综合信息 HUD 系统，旨在帮助玩家在复杂的 3D 空间中获得全方位的战场感知能力。

(1) 纳米能量护盾充能系统。坦克除了基础装甲生命值外，还配备了可循环充能的纳米能量护盾。当受到敌方炮火直接打击时，该护盾会自动吸收 70% 的伤害。如果在 3 秒内未受到任何新的攻击，护盾发生器将以每秒 15 点的速度自动为护盾重新充能直至满值。玩家可以充分利用地形掩体进行闪避和重整，大幅提升战局生存率。

(2) 侧后方红外威胁预警系统。由于第一人称视角存在视觉死角，系统内置了战场红外雷达预警。当有敌方单位绕至玩家视线范围外（左右夹角大于 55 度）且处于威胁距离内时，屏幕边缘会自动浮现半透明的红色警报三角指针，并实时计算并显示该敌人的具体物理距离。随着距离拉近，警报指针的颜色会变深且闪烁频次加快，指引玩家快速进行战略调整或反击。

(3) 旋转式高精雷达 Minimap。座舱右下方配备了全息极坐标扫描雷达。该雷达会以玩家当前的视线方向为基准，实时投影并旋转显示周围的砖墙、钢墙障碍物以及敌机位置。

## 武器系统与按键映射

操作者可以通过数字键切换不同的战术武器挂载，并利用鼠标调整视线方向进行精确打击。

键盘 W 键控制坦克沿视线方向前行，S 键向后退避，A 键与 D 键控制坦克相对于视线方向进行左、右侧向横移（Strafe）。使用鼠标移动可以平滑改变炮塔和主摄像机的旋转仰角。点击鼠标左键将发射当前选中的武器。

(1) 键盘数字键 1 切换为高爆动能弹 (AP Shells)，拥有无限备弹，是战场上的基础火力保障。

(2) 键盘数字键 2 切换为暴风散射弹 (Scatter Flak)，单次发射可呈扇形散出多发弹药，极适合清理近距离或行动敏捷的敌人。

(3) 键盘数字键 3 切换为纳米穿透激光 (Laser Beam)，能发射一道贯穿整条直线所有障碍物与敌机的高能光束。

(4) 键盘数字键 4 切换为磁暴休眠雷 (EMP Mine)，可在行进轨迹后方布置静止地雷，敌机触碰后会引发爆炸并伴随短暂的电磁瘫痪效果。

(5) 键盘数字键 5 切换为等离子冲撞 (Shield Ram)，激活后坦克将获得短暂的无敌屏障并向前高速冲刺，对沿途碰撞到的敌方单位造成毁灭性撞击伤害。
